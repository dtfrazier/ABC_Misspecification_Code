This folder contains the data and files necessary to recreate the figures for the first example in the supp. material

Files 

1) plots.m 

- This file is used to produce the first two figures in the supp. material. 

2) Accepted_data.mat

- This file contains the accepted draws of ABC-AR and ABC-Reg as well as the value of the misspecification parameter sigma

3) ABC_normal_compare.m

- This file contains the original file used to generate the results. This is equivalent to the R file in the Example_1_main folder
